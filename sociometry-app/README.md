# Sociometry Desktop App

## Kurulum

```bash
cd admin
npm install
npm run dev
```

## Google Credentials

`admin/.env` dosyasına aşağıdaki alanları ekleyin:

- `GOOGLE_SERVICE_ACCOUNT_EMAIL`
- `GOOGLE_PRIVATE_KEY`

Credentials yoksa uygulama açılır, veri çekme ekranında kontrollü hata mesajı gösterir.
